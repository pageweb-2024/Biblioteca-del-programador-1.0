PYTHON

HISTORIA DEL LENGUAJE
    ORIGENES

    Python fue creado a finales de la década de 1980 por Guido van Rossum en los Países Bajos. Su desarrollo comenzó como un proyecto personal durante su tiempo libre y se lanzó oficialmente por primera vez en 1991 como Python 0.9.0

    PRIMERA VERSION

    Python 0.9.0: En febrero de 1991, se lanzó la primera versión oficial, Python 0.9.0, que ya incluía muchas de las características fundamentales del lenguaje, como clases, manejo de excepciones y funciones.

EVOLUCION

    Python 2.0(2000)

    Python 2.0 fue lanzado en octubre de 2000 e introdujo características significativas como la recolección de basura, la capacidad de trabajar con Unicode y nuevas herramientas de programación. Durante esta época, se formó una comunidad activa en torno a Python, que contribuyó al desarrollo y mejora del lenguaje.

    Python 3.0(2008)

    Python 3.0, lanzado en diciembre de 2008, fue una revisión que rompió la compatibilidad con versiones anteriores. Este cambio se hizo para eliminar redundancias y mejorar la coherencia del lenguaje. Se introdujeron muchas mejoras, incluyendo la función print() como una función en lugar de una declaración, mejoras en la gestión de cadenas de texto y una sintaxis más clara.

    PRESENTE

    En enero de 2020, se retiró oficialmente Python 2, lo que impulsó a más desarrolladores a adoptar Python 3.

PORQUE ES IMPORTANTE

    Python radica en su simplicidad, versatilidad, y su capacidad para adaptarse a una amplia variedad de dominios, desde la web hasta la ciencia de datos y la inteligencia artificial.

    Versatibilidad

    Python es un lenguaje multipropósito que se utiliza en diversos campos, lo que lo hace muy versátil: 

  • Desarrollo web, Ciencia de datos, Inteligencia artificial y aprendizaje automático, Automatización 
    Soporte multipltaforma

    Python se puede ejecutar en diferentes sistemas operativos (Windows, MacOS, Linux) sin cambios significativos en el código. Esto lo hace conveniente para desarrolladores que trabajan en ambientes heterogéneos.   

    Potencial educativo

    Muchas instituciones educativas enseñan Python en cursos de programación introductoria debido a su facilidad de uso. Su claridad ayuda a los estudiantes a centrarse en aprender conceptos fundamentales de programación sin preocuparse demasiado por detalles complejos de la sintaxis.

CONCEPTOS BASICOS DEL LENGUAJE
    Sintaxis basica

    Es la forma clásica de comenzar con cualquier lenguaje de rogramación.

    Variables

    int: Enteros (números sin decimales) float: Números de punto flotante (decimales) str: Cadenas de texto bool: Valores booleanos (verdadero o falso).

    Operadores

    Aritméticos: +, -, *, /, ** (potencia), % (módulo). De comparación: ==, !=, >, <,>=, <=. Lógicos: and, or, not.

    Estructuras de control de flujo condicionales

    se utilizan para ejecutar una instrucción o un conjunto de instrucciones solo si se cumple una determinada condición. Las estructuras de control de flujo condicionales más comunes en Python son “if“, “elif” y “else“. 

    Bucle “for“

    El bucle “for” se utiliza para iterar sobre una secuencia de elementos, como una lista o una cadena de texto.  

    Bucle “while“

    El bucle “while” se utiliza para repetir una instrucción o conjunto de instrucciones mientras se cumpla una condición determinada.

    Estructuras de control de flujo de control de excepciones

    se utilizan para manejar errores y excepciones en un programa. Las estructuras de control de flujo de control de excepciones más comunes en Python son “try“, “except“, “finally” y “raise“. 

    Funciones 

    Las funciones son bloques de código que realizan una tarea específica. En Python, las funciones se definen utilizando la palabra clave “def“.

HERRAMIENTAS CON LAS QUE SE USA
    PyCharm

    Uno de los IDEs más populares para Python, con características como autocompletado, navegación de código, depuración, soporte para Django, y gestión de entornos virtuales. 

    Visual Studio Code

    Un editor de código ligero y altamente extensible con soporte para Python a través de una extensión, además de integración con herramientas de depuración y control de versiones.

    Spyder

    Un IDE enfocado en científicos de datos, que incluye herramientas como consola interactiva, editor de scripts y gráficos en tiempo real. 

    Jupyter Notebook

    Un entorno interactivo que permite escribir y ejecutar código en celdas, ideal para análisis de datos, visualización, y documentación. 

LENGUAJES Y TECNOLOGIAS QUE COMPLEMENTAN EL LENGUAJE
    C/C++

    Python se integra bien con C y C++ a través de bibliotecas como Cython o herramientas como SWIG. Esto permite escribir partes críticas de una aplicación en C/C++ para mejorar el rendimiento, mientras se mantiene la simplicidad de Python para el resto. 

    JavaScript

    En el desarrollo web, Python se utiliza en el backend con frameworks como Django o Flask, mientras que JavaScript se utiliza en el frontend (con tecnologías como React, Vue, o Angular). 

    Java

    Si bien Python es excelente para la creación rápida de prototipos, Java a menudo se utiliza en sistemas de backend grandes y escalables. Ambas tecnologías se pueden combinar a través de interfaces como Jython, que permite ejecutar código Python en la JVM (Máquina Virtual de Java).

RESUMEN

    En resumen, Python se complementa bien con varios lenguajes y tecnologías. 

      Se adapta a una amplia gama de tecnologías y lenguajes, lo que lo convierte en una opción poderosa y versátil para desarrollo web, ciencia de datos, automatización y más.
