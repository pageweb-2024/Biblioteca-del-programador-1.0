CSS

HISTORIA DEL LENGUAJE
    ORIGENES

    CSS fue propuesto por Håkon Wium Lie en 1994 mientras trabajaba en el CERN, y su propuesta fue publicada oficialmente en diciembre de 1996 por el W3C (World Wide Web Consortium).

    PRIMERA VERSION

    CSS1: La primera versión oficial, CSS1, fue publicada en diciembre de 1996. Introdujo reglas básicas de estilo para texto, colores, márgenes y fuentes. Se centraba en la presentación visual de documentos HTML.

    CSS2 (1998)

    CSS2 se publicó en mayo de 1998 y amplió las capacidades de CSS1. Incluyó soporte para posicionamiento, media types (como impresión y pantallas), y más selectores. También introdujo la propiedad z-index.

    CSS2.1 (2011)

    CSS2.1 fue una actualización y revisión de CSS2, publicada en junio de 2011. Se enfocó en corregir errores y mejorar la interoperabilidad entre navegadores.

    PRESENTE

    CSS3 no se lanzó como una versión única, sino que se desarrolló en módulos. Esto permitió la introducción de nuevas características de manera más rápida y flexible. Introdujo propiedades avanzadas como bordes redondeados, sombras, gradientes, transiciones y animaciones. También se mejoró el diseño responsivo mediante el uso de media queries.

PORQUE ES IMPORTANTE

    CSS (Cascading Style Sheets) es crucial en el desarrollo web por varias razones.

    Separación de Contenido y Presentación
    • Organización: Permite separar el contenido HTML (estructura y semántica) de la presentación (estilos), lo que facilita la gestión y el mantenimiento del código.
    • Flexibilidad: Puedes cambiar el diseño de una página sin modificar su contenido.
    Mejora la Usabilidad y la Accesibilidad
    • Diseño Adaptativo: CSS permite la creación de diseños responsivos que se ajustan a diferentes tamaños de pantalla, mejorando la experiencia del usuario en dispositivos móviles y tabletas.
    • Accesibilidad: Facilita la creación de sitios web más accesibles, permitiendo estilos que se adapten a las necesidades de los usuarios, como aumentar el contraste o el tamaño de la fuente.
    Control de Estilos
    • Personalización: Permite a los desarrolladores tener un control total sobre la apariencia de los elementos en una página web, desde colores y fuentes hasta márgenes y alineaciones.
    • Consistencia: Puedes aplicar estilos de manera uniforme en múltiples páginas, asegurando una experiencia de usuario coherente en todo el sitio.
CONCEPTOS BASICOS DEL LENGUAJE
    selectores
    • Por tipo: Selecciona todos los elementos de un tipo específico (ej. p para párrafos).
    • Por clase: Selecciona elementos con una clase específica (ej. .clase).
    • Por ID: Selecciona un elemento con un ID específico (ej. #id).
    • Selectores descendientes: Selecciona elementos que son descendientes de otros (ej. div p selecciona todos los párrafos dentro de un div).
    Propiedades y Valores
    • Propiedades: Son los aspectos que se pueden modificar (ej. color, font-size, margin).
    • Valores: Son los ajustes que se aplican a las propiedades (ej. red, 16px, 10px).
    Cascada y Especificidad
    • Cascada: Se refiere a cómo se aplican los estilos cuando hay múltiples reglas que podrían aplicarse a un mismo elemento. Los estilos se aplican en un orden específico: estilos en línea, estilos internos y estilos externos.
    • Especificidad: Es el mecanismo que determina qué regla CSS se aplica cuando hay conflictos. Se basa en la combinación de tipos de selectores (ID, clase, tipo).
    Modelo de Caja (Box Model)
    • Contenido: El área donde se muestra el texto y las imágenes.
    • Padding: Espacio entre el contenido y el borde (interno).
    • Borde: Línea alrededor del padding y del contenido.
    • Margen: Espacio exterior al borde (externo).
    Unidades de Medida
    • Unidades Absolutas: Se basan en una medida fija (ej. px, pt).
    • Unidades Relativas: Se basan en el contexto (ej. em, rem, %).
    Colores

      Métodos de especificación: Puedes definir colores de diferentes maneras:

    • Nombres de color: red, blue, etc.
    • Código HEX: #FF0000 para rojo.
    • RGB: rgb(255, 0, 0) para rojo.
    • RGBA: rgba(255, 0, 0, 0.5) para rojo con opacidad.
    Estilos de Texto

      Propiedades comunes:

    • font-family: Tipo de letra.
    • font-size: Tamaño de la letra.
    • font-weight: Grosor de la letra (normal, bold).
    • text-align: Alineación del texto (left, right, center).
    posicionamiento

      Estilos de posicionamiento:

    • static: El valor por defecto; los elementos se posicionan en el flujo normal del documento.
    • relative: Posiciona el elemento relativo a su posición original.
    • absolute: Posiciona el elemento en relación a su contenedor más cercano que no sea estático.
    • fixed: Posiciona el elemento relativo a la ventana del navegador.
    • sticky: Una combinación entre relative y fixed.
    Media Queries

      Definición: Se utilizan para aplicar diferentes estilos en función de las características del dispositivo, como el tamaño de la pantalla.

    Flexbox y Grid
    • Flexbox: Un modelo de diseño unidimensional que facilita la distribución del espacio entre elementos en una interfaz.
    • Grid: Un modelo de diseño bidimensional que permite crear layouts más complejos utilizando filas y columnas.
HERRAMIENTAS CON LAS QUE SE USA
    Visual Studio Code

    Un editor de código fuente popular y altamente personalizable con soporte para extensiones, que incluyen resaltado de sintaxis y autocompletado para CSS.

    Sublime Text

    Un editor de texto rápido y ligero que permite la edición de archivos CSS con múltiples funcionalidades.

    Atom

    Un editor de código open-source creado por GitHub, con una interfaz amigable y opciones de personalización.

LENGUAJES Y TECNOLOGIAS QUE COMPLEMENTAN EL LENGUAJE
    HTML

    Es el lenguaje de marcado que estructura el contenido en la web. HTML proporciona la semántica y la estructura básica, mientras que CSS se encarga de la presentación visual.

    JavaScript

    Es un lenguaje de programación que permite agregar interactividad y comportamiento dinámico a las páginas web.

    Lenguajes de Backend

    PHP, Python, Ruby, Java, Node.js: Lenguajes utilizados en el lado del servidor para gestionar la lógica de la aplicación, bases de datos, y el procesamiento de datos.

RESUMEN

    CSS es fundamental para cualquier persona interesada en el desarrollo web. CSS es esencial para crear páginas web efectivas y visualmente atractivas, además de ser un paso crucial en el camino hacia una carrera exitosa en el desarrollo web.
