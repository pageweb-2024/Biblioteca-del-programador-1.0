HISTORIA DE LA PROGRAMACIÓN WEB

HISTORIA

La historia de la programación web es un recorrido fascinante que se ha desarrollado desde los primeros días de Internet hasta las sofisticadas aplicaciones web que utilizamos hoy en día.

PRIMEROS AÑOS DE LA WEB(1990-1995)

  • HTML (1990):HTML (1990): La web como la conocemos comenzó con Tim Berners-Lee, quien en 1990 creó el HTML (Hypertext Markup Language), un lenguaje para estructurar contenido en la web. Berners-Lee también desarrolló el primer navegador web y servidor.

  • HTTP y URL:Junto con HTML, Berners-Lee introdujo el HTTP (Hypertext Transfer Protocol) y las URLs (Uniform Resource Locators), que permitieron la comunicación entre servidores y navegadores.

  • Netscape Navigator (1994):Uno de los primeros navegadores web populares fue Netscape Navigator, que ayudó a popularizar el acceso la web.

LA WEB DINAMICA(1995-2005)

  • JavaScript (1995):JavaScript (1995): Brendan Eich creó JavaScript en 1995 para hacer que las páginas web fueran interactivas. Este lenguaje de programación permitió a los desarrolladores crear efectos en tiempo real, como validaciones de formularios y menús desplegables.

  • CSS (1996):CSS (1996): Las CSS (Cascading Style Sheets) fueron introducidas por el W3C para separar el contenido de la presentación, permitiendo a los desarrolladores web controlar el estilo y el diseño de las páginas web de manera más eficiente.

  • PHP (1995):PHP, creado por Rasmus Lerdorf, es un lenguaje de programación del lado del servidor que facilitó la creación de contenido web dinámico y la interacción con bases de datos.

  • Flash (1996):Adobe Flash se hizo popular para crear animaciones, juegos y contenido interactivo en la web. Aunque fue muy utilizado, hoy en día ha sido reemplazado por tecnologías más modernas.

  • ASP.NET (2002):Microsoft lanzó ASP.NET, que permitió a los desarrolladores escribir código de servidor en lenguajes como C# o VB.NET.
