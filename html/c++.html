C++

HISTORIA DEL LENGUAJE
    ORIGENES

    C++ fue desarrollado por Bjarne Stroustrup en 1979 en los Laboratorios Bell como una extensión del lenguaje C. Stroustrup buscaba agregar características de programación orientada a objetos (POO) a C, lo que resultó en un lenguaje más flexible y potente.

    PRIMERA VERSION

    El primer compilador C++ apareció en 1985. Este lenguaje fue diseñado para ser un lenguaje multipropósito, eficiente y con un alto nivel de control sobre los recursos del sistema.

EVOLUCION

A lo largo de los años, C++ ha pasado por varias actualizaciones significativas. Los estándares más importantes incluyen:

    C++98 (1998):

    Primera gran estandarización del lenguaje.

    C++11 (2011):

    Introdujo características modernas como lambdas, punteros inteligentes y el bucle for basado en rangos.

    C++14, C++17 y C++20:

    Añadieron mejoras de rendimiento, nuevas librerías estándar y características avanzadas como módulos y conceptos.

PORQUE ES IMPORTANTE
    Rendimiento:

    C++ es conocido por ser uno de los lenguajes más rápidos y eficientes debido a su cercanía al hardware y su capacidad para gestionar directamente la memoria. Esto es crucial en áreas donde el rendimiento es fundamental, como los videojuegos, sistemas operativos y aplicaciones embebidas. 

    Control de recursos:

    A diferencia de lenguajes de alto nivel como Python, C++ permite un control detallado sobre cómo se gestionan los recursos del sistema, lo que es útil en entornos donde se requiere optimización extrema, como aplicaciones en tiempo real o simulaciones científicas.   

    Multiparadigma:

    Soporta tanto la programación estructurada (como en C) como la programación orientada a objetos (POO). Además, permite la programación genérica mediante plantillas, lo que lo convierte en un lenguaje muy flexible para una amplia gama de aplicaciones. interacción del jugador con el entorno.

    Extensa base de código:

    Muchas infraestructuras críticas, videojuegos AAA, motores gráficos (como Unreal Engine), navegadores web, y aplicaciones financieras están escritas o dependen de C++.

CONCEPTOS BASICOS DEL LENGUAJE
    Variables y tipos de datos:

    Como en la mayoría de los lenguajes, C++ trabaja con tipos de datos básicos como int, float, char, double, entre otros.

    Punteros y referencias:

    Una característica fundamental de C++ es el uso de punteros, que permiten acceder directamente a la memoria. Esto es poderoso, pero también puede ser propenso a errores si no se maneja con cuidado.

    Clases y objetos:

    En C++, las clases son fundamentales para la programación orientada a objetos. Las clases definen las propiedades y el comportamiento de los objetos, que son instancias de clases. Los conceptos de herencia, polimorfismo y encapsulación son claves en la POO de C++.

    Sobrecarga de funciones y operadores:

    C++ permite sobrecargar funciones (definir varias funciones con el mismo nombre, pero diferentes parámetros) y operadores, lo que facilita el trabajo con tipos de datos personalizados.

    Plantillas (Templates):

    C++ soporta programación genérica, lo que permite escribir funciones y clases que funcionan con cualquier tipo de dato, aumentando la reutilización del código.  

    Manejo de memoria dinámica:

    A través de new y delete, C++ permite asignar y liberar memoria durante la ejecución del programa, dándole un control preciso sobre el uso de los recursos.

    Excepciones:

    C++ tiene un sistema de manejo de errores mediante excepciones (try, throw, catch), lo que ayuda a hacer el código más robusto. 

    Sintaxis básica 
  • Comentarios: Se pueden hacer comentarios en una línea usando // o en varias líneas usando /* ... */
    Main Function 
  • El punto de entrada de un programa C++ es la función main
    Variables y Tipos de Datos 

    C++ es un lenguaje fuertemente tipado, lo que significa que cada variable debe tener un tipo definido.

    Tipos de datos básicos:

  • int: Números enteros.
  • float: Números en coma flotante de precisión simple.
  • double: Números en coma flotante de doble precisión.
  • char: Un carácter.
  • bool: Booleano (verdadero o falso).
    Operadores 
  • Aritméticos: +, -, *, /, % (módulo).
  • Relacionales: ==, !=, <, >, <=, >=.
  • Lógicos: && (AND), || (OR), ! (NOT).
  • Asignación: =, +=, -=, *=, /=, %=.
    Control de flujo 

    Condicionales: Se usan para tomar decisiones.

    Bucles: Se utilizan para repetir un bloque de código varias veces.

  • for: Bucle con un contador.
  • while: Bucle que se repite mientras la condición es verdadera
  • do-while: Similar a while, pero se ejecuta al menos una vez.
    Funciones

    Las funciones en C++ permiten reutilizar código y mejorar la organización. 

    Punteros 

    Un puntero es una variable que almacena la dirección de memoria de otra variable.

    Los punteros se usan ampliamente en C++ para manipular la memoria directamente.

    Clases y Objetos (Programación Orientada a Objetos)

    C++ es un lenguaje orientado a objetos. Las clases definen tipos de datos abstractos y los objetos son instancias de esas clases. 

    E/S (Entrada y Salida)

    Entrada: Se usa cin para leer valores.

    Salida: Se usa cout para imprimir valores.

    Espacios de nombres (Namespaces)

    En C++ se utilizan espacios de nombres para evitar colisiones de nombres. 

    Herencia y Polimorfismo 

    La herencia permite que una clase hija herede propiedades y comportamientos de una clase padre.

    El polimorfismo permite tratar objetos de diferentes clases derivadas de manera uniforme usando punteros o referencias de la clase base.

HERRAMIENTAS CON LAS QUE SE USA
    Compiladores:
  • GCC (GNU Compiler Collection): Compilador multiplataforma muy popular en entornos Unix y Linux.
  • MSVC (Microsoft Visual C++): Compilador de C++ de Microsoft, ampliamente utilizado en Windows.
  • Clang: Un compilador modular y rápido, utilizado en macOS y también disponible en otras plataformas.
    IDE (Entornos de desarrollo integrados):
  • Visual Studio: Uno de los IDEs más utilizados para C++ en Windows, con una gran cantidad de herramientas de depuración y análisis.
  • CLion: Un IDE moderno multiplataforma creado por JetBrains, muy apreciado por su compatibilidad con CMake y su soporte para C++.
  • Code: Blocks: Un IDE ligero y fácil de usar para proyectos más pequeños o educativos.
    Depuradores:
  • GDB: El depurador estándar en sistemas basados en Linux.
  • LLDB: Un depurador rápido que forma parte del proyecto LLVM, usado principalmente con Clang.
    Sistemas de construcción:
  • CMake: Una herramienta multiplataforma que facilita la configuración y gestión de proyectos C++ grandes y complejos
    Conocimientos previos recomendados
  • Lógica de programación: Como C++ es un lenguaje de nivel intermedio/alto, tener una base sólida en conceptos de lógica de programación es fundamental. Esto incluye control de flujo (condicionales, bucles), estructuras de datos (arrays, listas, pilas), y algoritmos básicos.
  • Conocimiento de C: Aunque no es obligatorio, C++ tiene muchas similitudes con C. Familiarizarse con C puede ayudarte a entender la evolución del lenguaje y a trabajar más cómodamente con C++.
  • Sistemas y memoria: C++ es conocido por su control sobre el hardware y la memoria. Entender cómo funciona la gestión de memoria (asignación dinámica, punteros) será esencial para sacar provecho de sus capacidades.
    Conceptos avanzados de C++ para explorar después
  • Punteros y memoria dinámica: Uno de los conceptos más poderosos y desafiantes de C++ son los punteros y la gestión de memoria dinámica (con new y delete). Aprender a manejarlos correctamente es crucial para evitar errores como los memory leaks.
  • Programación orientada a objetos (POO): C++ es conocido por soportar la POO. Aprender a trabajar con clases, herencia, polimorfismo y encapsulamiento es clave para estructurar bien el código y reutilizar componentes.
  • Templates y programación genérica: C++ ofrece templates, que te permiten escribir código que funcione con cualquier tipo de dato. Este concepto es muy útil para crear estructuras de datos y algoritmos genéricos que pueden trabajar con diferentes tipos sin duplicar el código.
  • STL (Standard Template Library): C++ incluye la poderosa STL, que es una colección de estructuras de datos y algoritmos predefinidos como vectores, listas, colas, y más. Entender cómo usarla te ahorrará tiempo y esfuerzo.
    Frameworks y herramientas para aprender C++

    IDE y compiladores:

  • Visual Studio: Un IDE robusto que soporta bien C++ y facilita la depuración de código.
  • Code::Blocks: Un IDE ligero que es popular para el desarrollo en C++.
  • CLion: Un IDE avanzado de JetBrains para C++, con herramientas de análisis de código.
  • GCC (GNU Compiler Collection): Uno de los compiladores más utilizados para C++.

    • CMake: Una herramienta de automatización de compilaciones que es muy útil para proyectos C++ grandes y complejos. Facilita la generación de archivos de construcción en diferentes entornos.

    Depuradores: Aprender a utilizar herramientas de depuración como gdb (GNU Debugger) será esencial para identificar errores y mejorar tu código.

LENGUAJES Y TECNOLOGIAS QUE COMPLEMENTAN EL LENGUAJE
  • C: Dado que C++ es una extensión de C, ambos lenguajes son completamente interoperables. El código C puede integrarse fácilmente dentro de proyectos C++.
  • Python: Aunque no está tan cerca del hardware como C++, Python a menudo se usa junto con C++ para desarrollar herramientas de scripting y automatización, especialmente en videojuegos y aplicaciones científicas. Las bibliotecas como Boost.Python permiten la integración de ambos lenguajes.
  • OpenGL/Vulkan: Para la programación gráfica, C++ a menudo se utiliza con APIs como OpenGL o Vulkan para la creación de gráficos 2D/3D, comúnmente en videojuegos o simulaciones.
  • Assembly: Para tareas de optimización extrema, especialmente en sistemas embebidos o desarrollo de drivers, tener conocimientos básicos de Assembly puede complementar tu uso de C++.
  • OpenGL o DirectX: Si estás interesado en el desarrollo de gráficos y videojuegos, C++ se usa junto con librerías como OpenGL o DirectX para la creación de gráficos 2D/3D.
  • Bases de datos: Aunque C++ no tiene una conexión directa con bases de datos como lenguajes web, puedes interactuar con bases de datos usando bibliotecas como SQLite, MySQL, o PostgreSQL.
  • Librerías de terceros:
    • Boost: Una colección de librerías que extienden las capacidades estándar de C++, ofreciendo herramientas para trabajar con datos, expresiones regulares, punteros inteligentes, etc.
    • Qt: Un framework multiplataforma utilizado para desarrollar aplicaciones gráficas. Es muy popular en el desarrollo de interfaces gráficas de usuario (GUI) con C++.
    • Unreal Engine: Un motor de videojuegos ampliamente utilizado que está basado en C++.
    Proyectos y práctica:

    La práctica es fundamental para aprender C++. Aquí algunas ideas de proyectos que te ayudarán a mejorar:

  • Estructuras de datos: Implementa tus propias versiones de estructuras como listas enlazadas, árboles, pilas, y colas para entender mejor el uso de punteros y memoria.
  • Sistemas embebidos: Si te interesa el desarrollo de software cercano al hardware, C++ es ideal para trabajar en dispositivos embebidos (Arduino, Raspberry Pi).
  • Juegos: Usa un motor de juegos como Unreal Engine para crear tus propios juegos en 3D o 2D. También puedes desarrollar un juego simple en consola para entender la lógica básica de un juego.

RESUMEN

    C++ es uno de los lenguajes más potentes y eficientes, utilizado en una amplia gama de aplicaciones que van desde videojuegos hasta sistemas operativos y software embebido. Su historia está profundamente vinculada a C, con el objetivo de proporcionar un lenguaje que combine eficiencia y control sobre el hardware con características avanzadas de la programación orientada a objetos. Aunque tiene una curva de aprendizaje empinada debido a su complejidad, su capacidad para manejar recursos de manera eficiente lo hace ideal para aplicaciones donde el rendimiento es crucial. Complementado con otras tecnologías como C, Python, OpenGL, y librerías como Boost, C++ sigue siendo uno de los lenguajes más relevantes en la programación moderna. Para aprender C++, necesitarás practicar conceptos avanzados como la gestión de memoria y la programación orientada a objetos, usar herramientas como compiladores y depuradores, y complementar tu conocimiento con tecnologías y bibliotecas que te permitan desarrollar proyectos más grandes y complejos.
