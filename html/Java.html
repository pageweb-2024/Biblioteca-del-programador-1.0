JAVA

HISTORIA DEL LENGUAJE
    ORIGENES

    Java comienza a principios de los años 90 en Sun Microsystems, una empresa de tecnología fundada por Andy Bechtolsheim, Bill Joy, Vinod Khosla y Scott McNealy. El desarrollo de Java fue liderado por James Gosling, quien es considerado el "padre" de Java.

    PRIMERA VERSION

    En 1991, un equipo de ingenieros de Sun Microsystems, liderado por James Gosling, inició el proyecto denominado "Green Project". El objetivo original era desarrollar una tecnología que pudiera funcionar en electrodomésticos y pequeños dispositivos como televisores y decodificadores. El equipo creó un lenguaje llamado Oak, inspirado en lenguajes como C y C++, pero diseñado para ser independiente de la plataforma. En 1994, al ver el crecimiento de la web, el equipo se dio cuenta de que Oak podría ser ideal para aplicaciones web interactivas. Cambiaron el enfoque hacia este nuevo mercado. Finalmente, en 1995, Oak fue renombrado como Java (nombre que proviene de un tipo de café) y lanzado al público con la versión 1.0 del JDK (Java Development Kit).

    EVOLUCION
    • Java 1.0 y la Web: Con el lanzamiento de Java en 1996, el lenguaje comenzó a ganar popularidad rápidamente. Sun Microsystems promovió Java como un lenguaje que permitía a los desarrolladores escribir código una sola vez y ejecutarlo en cualquier plataforma que tuviera la Máquina Virtual de Java (JVM).
    • Java en código abierto: En 2006, Sun Microsystems decidió hacer de Java un software de código abierto bajo la licencia GNU General Public License (GPL). Este movimiento fue un paso importante en la historia de Java, ya que permitió a la comunidad contribuir activamente a su desarrollo.
    PRESENTE

    Desarrollo continuo y presente: Desde 2018, Oracle adoptó un ciclo de lanzamientos de versiones cada seis meses para Java. Este ciclo garantiza actualizaciones frecuentes y permite a los desarrolladores adoptar nuevas características de manera constante. Las versiones Java 11 y Java 17 son versiones de soporte a largo plazo (LTS), utilizadas ampliamente en proyectos empresariales debido a su estabilidad.

PORQUE ES IMPORTANTE

Java es importante por varias razones clave, que han contribuido a su impacto duradero en el mundo de la tecnología y su adopción masiva en múltiples industrias.

    Portabilidad (Write Once, Run Anywhere - WORA)

    Java fue diseñado con la idea de ser independiente de la plataforma, lo que significa que el código escrito en Java puede ejecutarse en cualquier dispositivo que tenga una Máquina Virtual de Java (JVM).

    Popularidad en el desarrollo empresarial

    Java es el lenguaje dominante en el desarrollo de aplicaciones empresariales, principalmente debido a su plataforma Java EE (Enterprise Edition), ahora conocida como Jakarta EE.

    Seguridad

    Desde el principio, Java fue diseñado con un fuerte enfoque en la seguridad, especialmente para entornos de red.

CONCEPTOS BASICOS DEL LENGUAJE
    Sintaxis basica

    Es la forma clásica de comenzar con cualquier lenguaje de programación.

    Variables

    Tipos primitivos: Son tipos básicos y no objetos.

    • int: enteros (números sin decimales).
    • double: números con decimales.
    • char: caracteres individuales.
    • boolean: valores lógicos (true o false).
    Variables

    Tipos de referencia: Son tipos complejos como clases, arreglos y objetos.

    • String: cadenas de texto.
    • ArrayList: lista de objetos.
    Operadores
    • Aritméticos: +, -, *, /, ** (potencia), % (módulo).
    • De comparación: ==, !=, >, <,>=, <=.
    • Lógicos: and, or, not.
    • De asignación: =, +=, -=, *=, /=.
    Estructuras de control de flujo de control de excepciones

    El manejo de errores permite capturar y tratar los errores que ocurren durante la ejecución de un programa para que no se detenga abruptamente. En JAVA, existen diversas formas de manejar errores, incluyendo el uso de la función try-catch para manejar excepciones.

    Bucle “for“

    El bucle “for” se utiliza para iterar sobre una secuencia de elementos, como una lista o una cadena de texto.

    Bucle “while“ y “do-while“
    • El bucle “while” se utiliza para repetir una instrucción o conjunto de instrucciones mientras se cumpla una condición determinada.
    • es similar al bucle while, pero garantiza que el bloque de código se ejecute al menos una vez, ya que la condición se evalúa después de la ejecución.
    Switch

    Es útil cuando tienes múltiples posibles valores para una variable se inicia con el “Switch”.

HERRAMIENTAS CON LAS QUE SE USA
    Eclipse

    Uno de los IDEs más populares y versátiles para Java. Tiene soporte para plugins y es ampliamente utilizado en el desarrollo empresarial.

    Javac

    El compilador que convierte el código fuente en bytecode que se ejecuta en la Máquina Virtual de Java (JVM).

    Maven

    Es la herramienta de gestión de dependencias más popular en el ecosistema Java. Facilita la construcción, empaquetado y configuración de proyectos, utilizando un archivo pom.xml para definir dependencias y configuraciones del proyecto.

LENGUAJES Y TECNOLOGIAS QUE COMPLEMENTAN EL LENGUAJE
    JavaScript

    JavaScript es el principal lenguaje para el desarrollo web del lado del cliente, y frameworks como Node.js también permiten su uso en el backend, aunque a menudo se usa en combinación con Java en arquitecturas de microservicios.

    Python

    Aunque Python no se ejecuta directamente sobre la JVM, se puede integrar con Java a través de Jython, una implementación de Python para la JVM..

    HTML y CSS

    En aplicaciones web basadas en Java, como las que usan Spring o JavaServer Faces (JSF), el frontend utiliza HTML para estructurar el contenido y CSS para diseñarlo.

RESUMEN

Java se complementa con varios lenguajes y tecnologías para desarrollar aplicaciones completas y eficientes. Java, junto a estas tecnologías, ofrece un ecosistema amplio para construir aplicaciones web, móviles y empresariales escalables y modernas.
